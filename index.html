How it works

Paste or drop. Detect. Review and export.

From raw input to clean output in three steps.

1

Paste or drop

Paste text directly or drag-and-drop files. Supported formats: PDF, DOCX, XLSX, TXT, MD, CSV. Process one or batch multiple files.

2

Detect locally

Two local models identify sensitive entities: one for international PII, one for Italian documents. Both run on your machine.

3

Review and export

Review detected entities, restore false positives, add manual redactions. Export clean text as TXT or Markdown.

Capabilities

Six capabilities. Zero cloud dependency.

Built for developers, analysts, and anyone who shares sensitive text with AI services.

01

Text Redaction

Paste or type text directly. PII is detected and replaced in real-time with your chosen redaction style.

02

Document Import

Drag-and-drop PDF, DOCX, XLSX, TXT, MD, or CSV files. Text is extracted and processed locally.

03

Batch Processing

Queue multiple files for processing. Export results as TXT or Markdown.

04

Entity Review

Interactive post-redaction panel. Accept or reject detected entities, add manual redactions, fine-tune results.

05

Dual AI Models

Two local models: one for international PII, one for Italian documents. Switchable from Settings.

06

HTTP API + CLI

Local REST API with OpenAPI docs for integration. Standalone CLI binary for headless automation.

Architecture

Built local-first, top to bottom

No remote inference. No hidden backend. The app, models, parser and API run locally on your machine.
